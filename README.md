# MonsterGame
## 몬스터 잡기 게임
> Java Swing과 멀티스레드를 활용한 간단한 GUI 기반 게임입니다.
> 플레이어는 화면에 나타나는 공격 몬스터와 방어 몬스터를 마우스로 클릭하여 처치해야 하며, 플레이어의 체력이 0이 되면 게임이 종료됩니다.

## 주요 기능
- 공격 몬스터
  * 빨간색 배경 `(╬ಠ益ಠ)`
  * 5초마다 플레이어 공격 -> HP 감소
  * 한 번 클릭으로 처치 가능
  * 몬스터가 죽을수록 전체 공격 데미지가 점점 감소
- 방어 몬스터
  * 파란색 배경 `(ಠ_ಠ)`
  * 더블 클릭으로 처치 가능
  * 단순 이동 및 방어 행동 수행
 
## 게임 규칙
- 체력 : 초기값 100, 공격 몬스터에게 5초마다 피해 입음
- 처치 방식 : 공격 몬스터 - 클릭 / 방어 몬스터 - 더블클릭
- 승리 조건 : 20마리 몬스터를 모두 처치
- 패배 조건 : 플레이어 체력 0
